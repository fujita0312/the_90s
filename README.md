## 🚀 Getting Started

### Prerequisites
- Node.js 16+ 
- Yarn package manager (recommended)
- Modern web browser

### Installation

1. **Clone the repository**
   ```bash
   git clone <repository-url>
   cd 90s-fresh/frontend
   ```

2. **Install Dependencies**
   ```bash
   yarn install
   # or
   npm install
   ```

3. **Start Development Server**
   ```bash
   yarn start
   # or
   npm start
   ```

4. **Open Browser**
   Navigate to `http://localhost:3000`

### Backend Setup (Optional)
If you want to use the guestbook and memes features:

1. **Navigate to server directory**
   ```bash
   cd ../server
   ```

2. **Install server dependencies**
   ```bash
   yarn install
   # or
   npm install
   ```

3. **Start the backend server**
   ```bash
   yarn dev
   # or
   npm run dev
   ```

4. **Backend will run on** `http://localhost:5003`
